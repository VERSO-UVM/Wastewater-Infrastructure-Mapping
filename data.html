Data Structure, Fields, and Collection

Overview

The Wastewater Infrastructure Mapping project involves the systematic collection and organization of data related to wastewater systems across towns in Vermont. This section provides detailed information about the data structure, the fields included, and the methods used for data collection.

Data Structure

The data is organized in a structured format to ensure consistency, ease of access, and usability. There are five datasets covering point, line and polygon data types. Each set of data is broken into towns.

Sewer Service Area (File name: townnameServiceArea.geojson)

  • System_Name - Wastewater system name
  • System_Owner - Wastewater system owner
  • TownID - Town identifier
  • Treatment_Facility - Treatment Facility name
  • GIS_Notes - Notes about data
  • GIS_Date - When the GIS data was originally created
  • GIS_Update - When the data was last updated
  • Creator - Default to VTANR and it will display VERSO if new data was added

Example: "properties":{"OBJECTID":0,"System Name":"Cold Brook Fire District","System Owner":"Cold Brook Fire District","TownID":25105,"Treatment Facility":"3-1296","GIS Notes":"interpreted from pipes, parcels, buildings, topo, as area served","GIS Date":"1993","GIS Update":"1993","Creator":"VTANR","Shape_Length":0,"Shape_Area":0}

Wastewater Infrastructure (Linear Features) (File name: townnameLinearFeatures.geojson)

  • Type - Describes the type of line, including stormline, sanitary, combined sewer, etc
  • Status - Existing, Proposed, Abandoned
  • Comment - Comment on the data
  • Source - Where data originated
  • MapDate - Date of map where data originated
  • Audience - Serving public, private
  • SystemType - Type of water system
  • Creator - Default to VTANR, change to VERSO if we create new data

Example data: "properties":{"OBJECTID":0,"Comment":"From Brattleboro Town","Map_Date":"","Audience":"Public","System_Type":"Wastewater","Type":"Sanitary Line","Status":"Existing","Source":"Contractor GIS/GPS","Creator":"VTANR","Shape_Length":0}

Wastewater Infrastructure (Point Features)(File name: townnamePointFeatures.geojson)

  • Type - Sanitary Manhole, Stormwater Manhole, Combined Sewer Manhole, CSO Outflow, etc
  • Status - Existing, Proposed, Abandoned
  • Comment - Comment on the data
  • Source - Where data was obtained from
  • MapDate - Date of map where data originates
  • SystemType - Type of water system
  • Creator - Default to VTANR, change to VERSO if we create new data

Example data: "properties":{"OBJECTID":0,"Comment":"From Brattleboro","Map_Date":"","System_Type":"Wastewater","Type":"Sanitary Manhole","Source":"Contractor GIS/GPS","Status":"Existing","Creator":"VTANR"}

Wastewater Facilities (File name: townnameWWTF.geojson)

  • FacilityName - Name of facility
  • DesignHydraulicCapacityInMGD - Amount of water the facility can process in a year
  • SeptageReceivedAtThisFacility - Does this facility receive septage? (Yes or no)
  • PermitID - PermitID
  • PermitRecordID - Permit Record ID
  • PermitteeName - Name of town served
  • ProgramCategory - Municipal, industrial, or pretreatment
  • NPDESPermitNumber - NPDES Permit Number
  • PermitLink - Link to permit

Example data: "properties":{"OBJECTID":0,"FacilityID":"174","FacilityName":"FiberMark North America Inc","DesignHydraulicCapacityInMGD":"2.00","SeptageReceivedAtThisFacility":"N","PermitID":"3-1136","PermitRecordID":"85","PermitteeName":"FiberMark North America Inc","ProgramCategory":"Industrial Discharge","NPDESPermitNumber":"VT0000248","PermitLink":"https://anrweb.vt.gov/ANR/vtANR/DocBrowser.aspx?Directory=DEC/WSMD/Wastewater/permits/3-1136/"}

Town Boundary (File name: townnameBorder.geojson)

  • GEOID - GEOID
  • NAME - Name of town
  • NAMELSAD - Name LSAD
  • LSAD - LSAD
  • MunicipalWastewater - Does the town have a municipal wastewater system? (Yes, No, or In Progress)

Example data: "properties":{"OBJECTID":223,"GEOID":"5002507900","Name":"Brattleboro","Name_LSAD":"Brattleboro town","LSAD":"43","Municipal_Wastewater":"Yes","Shape_Length":0,"Shape_Area":0}

Onsite Sewage Disposal Soil Ratings

  • Rating - Well Suited, Moderately Suited, Marginally Suited, Not Suited, Not Rated
