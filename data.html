Data Structure, Fields, and Collection

Overview

The Wastewater Infrastructure Mapping project involves the systematic collection and organization of data related to wastewater systems across towns in Vermont. This section provides detailed information about the data structure, the fields included, and the methods used for data collection.

Data Structure

The data is organized in a structured format to ensure consistency, ease of access, and usability. The primary data structure used is a relational database, which allows for efficient querying and management of the data.

Sewer Service Area

  • System_Name - Wastewater system name
  • System_Owner - Wastewater system owner
  • TownID - Town identifier
  • Treatment_Facility - Treatment Facility name
  • GIS_Notes - Notes about data
  • GIS_Date - When the GIS data was originally created
  • GIS_Update - When the data was last updated
  • Creator - Default to VTANR, change to VERSO if we create new data

Wastewater Infrastructure (Linear Features)

  • Type - Stormline, sanitary, combined sewer, etc
  • Status - Existing, Proposed, Abandoned
  • Comment - Comment on the data
  • Source - Where data was obtained from
  • MapDate - Date of map where data originates
  • Audience - Public, private
  • SystemType - Type of water system
  • Creator - Default to VTANR, change to VERSO if we create new data

Wastewater Infrastructure (Point Features)

  • Type - Sanitary Manhole, Stormwater Manhole, Combined Sewer Manhole, CSO Outflow, etc
  • Status - Existing, Proposed, Abandoned
  • Comment - Comment on the data
  • Source - Where data was obtained from
  • MapDate - Date of map where data originates
  • SystemType - Type of water system
  • Creator - Default to VTANR, change to VERSO if we create new data

Wastewater Facilities

  • FacilityName - Name of facility
  • DesignHydraulicCapacityInMGD - Amount of water the facility can process in a year
  • SeptageReceivedAtThisFacility - Does this facility receive septage? (Yes or no)
  • PermitID - PermitID
  • PermitRecordID - Permit Record ID
  • PermitteeName - Name of town served
  • ProgramCategory - Municipal, industrial, or pretreatment
  • NPDESPermitNumber - NPDES Permit Number
  • PermitLink - Link to permit

Town Boundary

  • GEOID - GEOID
  • NAME - Name of town
  • NAMELSAD - Name LSAD
  • LSAD - LSAD
  • MunicipalWastewater - Does the town have a municipal wastewater system? (Yes, No, or In Progress)

Onsite Sewage Disposal Soil Ratings

  • Rating - Well Suited, Moderately Suited, Marginally Suited, Not Suited, Not Rated
